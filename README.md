# Spninad Tap

## How do I install these formulae?

`brew install spninad/tap/<formula>`

Or `brew tap spninad/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "spninad/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
